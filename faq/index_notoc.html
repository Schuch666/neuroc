What is Neuroconductor?

Neuroconductor is a GitHub-based repository of R packages generally related to neuroimaging analyses and processing. The goals for the community are:

  1. Centralize these packages in one place
  2. Provide detailed howtos and tutorials (below) for users to easily set up and start using these packages
  3. Allow for more feedback for feature requests and contributions to these packages

The goals for the package developers are:

  1. Provide feedback on packages and features
  2. Help developers pass R CMD check with more detailed feedback and pull requests
  3. Check packages in a system with 3rd party imaging software (e.g.Â FSL, Freesurfer, AFNI, etc.) installed

What operating systems are supported?

The tutorials have been tested for Unix-based systems, specifically Linux and Mac OSX. Although we do not currently support it, Windows 10 has a Linux subsystem, which may allow for a working installation of all the dependencies of Neuroconductor.

Although still untested there is a good tutorial on running FSL on Windows as well as ANTsR on Windows.

Who developed Neuroconductor?

John Muschelli, Jean-Philippe Fortin, Adi Gherman, Brian Caffo and Ciprian Crainiceanu all have contributed a significant amount of time to the project.

Where can I get the code for the tutorial/howto?

All code for the tutorials and installation guides can be located at https://github.com/muschellij2/neuroc. Pull requests are welcome. If you would like a specific tutorial or content, please open an issue.

Where should I start?

The workflow is that one would go through Installation Guides, then Required Readings, and then to any specific Tutorial that a reader would like to learn about.

What if something doesn't work?

Please open an issue on GitHub.
